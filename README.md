# Jeevithan R R — Portfolio

Personal portfolio website for Jeevithan R R, AI Engineer & Full Stack Developer.

## Structure

```
portfolio/
├── index.html   # Main HTML
├── style.css    # All styles
├── script.js    # Interactive JS (Three.js 3D, animations, cursor)
└── README.md
```

## Deploy on Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Framework Preset: **Other** (static HTML)
5. Click **Deploy** — no build settings needed

## Local Development

Just open `index.html` in your browser — no build step required.
